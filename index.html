  • Service: Architecture and software development.

  • Service: Committer, PMC Chair

Domain Driven Design using Naked Objects, Restful Objects

Dan Haywood has written or co-written several books. DDD using NO explained how the Naked Objects framework (now Apache Isis) could be used to build applications from domain classes. The Restful Objects spec described how a domain application can be made available over a hypermedia API.

  • Client: Pragmatic Programmers.
  • Date: 2009, 2011
  • Service: Technical author
